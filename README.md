# StructureDataD-Pertemuan1
1.Pengertian konsep array
Array adalah salah satu struktur data yang sangat penting di era modern hingga sekarang, dari pemula sampai profesional, programer menggunakan array untuk menyimpan dan mengolah sebuah data dalam jumlah yang besar secara efisien. Array adalah sekumpulan data dengan tipe yang sama dalam satu variabel, pada setiap array memimiliki massing-masing indeks sehingga mudah untuk diakses.
